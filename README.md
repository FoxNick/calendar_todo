# calendar_todo
任务todo 日历 农历算法源自许剑伟先生的寿星万年历。
万年历的代码移植自HongchenMeng先生的c#移植项目，地址：https://github.com/HongchenMeng/SharpSxwnl
感谢许剑伟先生！感谢HongchenMeng先生!

学习项目，比较粗糙，还在改进中。
task todo部分是想模仿org mode的，但是目前只实现了最基本的增删改。
calendar部分可以剥离出来单独使用。
